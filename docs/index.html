Build (HTML)

Pré-requis : Node.js.

  1. Installer les dépendances à la racine :
    • npm install
  2. Générer le HTML dans docs/ :
    • npm run slides:build
  3. Lancer un serveur Marp (optionnel) :
    • npm run slides:watch
